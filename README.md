# FlowChat 💬

Sistema de mensagens em terminal inspirado no WhatsApp, com foco em conceitos de Programação Orientada a Objetos (POO).

---

## 📌 Descrição

O **FlowChat** é uma aplicação desenvolvida em Python que simula um sistema de comunicação entre usuários, permitindo chats individuais e em grupo, além de diversas funcionalidades interativas.

O projeto foi construído com o objetivo de aplicar conceitos como herança, polimorfismo e organização modular do código.

## 🚀 Funcionalidades

- Autenticação com email (cadastro e login)
- Chat individual
- Chat em grupo
- Mensagens fixadas
- Mensagem temporária
- Reações em mensagens
- Menção coletiva (@todos)
- Enquetes
- Eventos
- Busca de mensagens

---

## 💡 Considerações

O FlowChat foi desenvolvido como um projeto acadêmico com o objetivo de simular funcionalidades de aplicativos de mensagens modernos em um ambiente simples de terminal.

Sua estrutura modular permite fácil expansão e adaptação para sistemas mais complexos no futuro.
